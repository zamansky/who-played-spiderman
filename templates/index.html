Using data to provide answers

Enter a query. It must start with who or when. When you submit the query, the server will find about 100 pages and look for the most common names (for who) or dates (for when). Names are defined as two words each capitalized and dates are in the form Month Day, Year, so Oct 10, 1955 is valid but then so is Thomas 22, 1234. This isn't meant to be perfect, just to show how using lots of data can get interesting and useful answers.

Try these, some work, some don't;

  • who played thor
  • who shot Ronald Reagan
  • when was D-Day
  • when was the Yom Kippur war
  • when was Elvis born
But when it doesn't work, if you look at the "guesses," the answer is generally there. The biggest problem is that we don't identify names well.

Note: This is slow and will probably crash at times
